# ResolvAI — Automated Customer Support Bot

FlowZint AI Hackathon 2026 | Support Chat Category

An AI-powered customer support chatbot for ShopNest e-commerce,
built with Python, Gradio 6, and OpenAI GPT-4o mini.

## Features
- Two-factor identity verification (Order ID + email)
- Order tracking and delivery status
- Refund eligibility checks
- Human agent escalation
- Scope enforcement

## Tech Stack
- Python 3
- Gradio 6
- OpenAI GPT-4o mini

## Setup
```bash
pip install gradio openai
export OPENAI_API_KEY="sk-..."
python resolveai.py
```
Then open: http://localhost:7860
